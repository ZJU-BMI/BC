# DAPP for medical process

## install guide

If you havn't installed truffle, run
```shell
npm install -g truffle
```

install dependencies

```shell
npm install
```

compile all contracts

```shell
truffle compile
```

test contracts

```shell
truffle test
```

build project
```shell
npm build
```

run server 

```shell
npm start
```
